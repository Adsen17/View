SYS-001 // SAAS

OrbitFlow — Gravity Task Manager

"A task manager that moves with your momentum."

OrbitFlow adalah aplikasi SaaS task manager modern dengan pendekatan unik berbasis physics (gravity-inspired interaction). Project ini dirancang untuk menghadirkan pengalaman berbeda dari task manager biasa dengan menggabungkan UI premium, interaksi dinamis, dan konsep visual yang terinspirasi dari dunia fisika.

CORE FEATURES
  • Gravity Mode: Elemen UI jatuh dan berinteraksi menggunakan physics engine.
  • Black Hole Delete: Hapus task dengan animasi tersedot ke dalam void.
  • Interactive Kanban: Drag & drop dengan animasi smooth dan responsif.
  • Deadline Orbit: Task divisualisasikan seperti planet yang mengorbit.
  • Modern Dashboard: Analytics terintegrasi menggunakan Recharts.
TECH STACK
Next.js TypeScript Tailwind CSS Zustand Framer Motion Matter.js dnd-kit
IN PROGRESS STATUS
Project ini masih dalam tahap pengembangan. Saat ini menggunakan data lokal (demo mode) dan belum terhubung ke database. Ke depannya akan diintegrasikan dengan Supabase untuk backend.
HIGHLIGHT: Advanced frontend engineering & cinematic animations.

SYS-002 // B2B

Shadow IT Tracker (SIT)

"Secure access, software inventory, & smart offboarding."

Shadow IT Tracker adalah platform SaaS internal (B2B) yang dirancang untuk membantu tim IT dan HR melacak, mengelola, dan mengamankan penggunaan aplikasi pihak ketiga (SaaS) yang tidak terdaftar oleh karyawan (Shadow IT). Aplikasi ini berfokus pada keamanan akses, manajemen inventaris perangkat lunak, dan alur kerja offboarding yang aman dengan antarmuka Neubrutalism yang tegas.

CORE FEATURES
  • Role-Based Access Control: Keamanan berbasis cookie & RLS untuk Manager dan Admin.
  • SaaS Inventory: Pusat kendali alat pihak ketiga, biaya, risiko, dan bulk delete.
  • Smart Offboarding: Pencabutan akses otomatis dengan checklist interaktif.
  • Immutable Audit Trail: Log aktivitas anti-ubah dengan Trigger database PostgreSQL.
  • Analytics Dashboard: Metrik real-time ekosistem IT dan estimasi pengeluaran.
TECH STACK
Next.js React TypeScript Tailwind CSS Supabase PostgreSQL Zod
HIGHLIGHT: Neubrutalism UI & Fullstack Backend Architecture.
STATUS: DEPLOYED
